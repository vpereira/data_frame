DataFrame is a flexible container for Matlab/R style data analysis in Ruby.

Its API is still at a very early stage of development.

Examples and motivation: http://www.lukasbiewald.com/?p=112

Home page: http://dataframe.rubyforge.org.  SVN: http://dataframe.rubyforge.org/svn/
